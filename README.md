# Project Overview
Smart-AI-Resume-Screening-System is an AI-powered recruitment solution designed to automate and optimize the resume screening process. The system uses Natural Language Processing (NLP), keyword extraction, and predictive scoring algorithms to match resumes with job descriptions and rank candidates efficiently.
This project demonstrates how AI significantly reduces manual screening time while improving hiring accuracy and consistency.

# Objectives
Automate resume shortlisting
Reduce manual screening effort
Improve candidate-job matching accuracy
Rank candidates based on relevance score
Streamline recruitment workflow

# Technologies Used
Python
NLP (Natural Language Processing)
TF-IDF / Keyword Matching
Cosine Similarity
Machine Learning (optional scoring model)
Jupyter Notebook

# How It Works
Upload resumes (PDF/Text format)
Input Job Description (JD)
Text preprocessing (tokenization, stopword removal)
Feature extraction using TF-IDF
Similarity calculation between resume & JD
Generate ranking scores
Display top candidates

# Traditional vs AI Screening

AI Screening	< 30 minutes	Consistent	Highly Scalable
